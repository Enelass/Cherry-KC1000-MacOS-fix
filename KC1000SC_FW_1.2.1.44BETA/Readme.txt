Preconditions:
•	Run the update on a Windows 10 64 bit
•	KC1000SC Driver 1.0.5.xxx installed on system
	-->In case the driver is not available, a readable Smartcard must be inserted, when starting the *.bat file




Update Steps:
•	Plug in Keyboard
•	Execute “Start_firmware_uploadl_1_2_x_xxx.bat”
•	Comandline shows:
Connecting to Cherry KC 1000 SC 0
###############################################################################################################################################################################################################################################################################################################################################################################################################################################################################
•	… now the firmware is updated
•	Press any key to close window
•	Unplug Keyboard
